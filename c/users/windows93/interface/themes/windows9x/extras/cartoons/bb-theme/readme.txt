Beavis & Butt-head Desktop Theme v2.0
-------------------------------------
                      by Adrian Pauly


Huh, huh.. this theme rules!!

You get 2 wallpapers, 2 color schemes, original sounds, cursors, 
icons, startup and shutdown screens, and a really cool original
screen saver!

Just copy the files to whatever directory you wish to have 
the theme in, then copy the font (COMIC_O.TTF)
to your Windows\Fonts directory, and run the BEAVIS.BAT 
file to change the shutdown screens.  When you wish to
remove the theme, run RESTORE.BAT to restore your original
startup/shutdown screens.
To install the screen saver, run the setup.exe file included in 
this directory, and install it to your \Windows directory.
