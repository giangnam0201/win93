************ The Rocky and Bullwinkle Windows 95 Plus! Theme ************ 


960125 version

For this to work correctly, you need to unzip this file in your
"Themes" directory.  The file "Bullwinkle (high color).theme" will go
in the "Themes" directory while all other files will be unzipped in
a "Themes\Bullwinkle" directory.  Also please note that by the "Themes"
directory I mean whatever directory all your Plus! Themes are in.

These files contain long file names, so make sure that the names are 
maintained when unzipped.

This theme also contains the icons that I've assembled in the file
ff_icons_??????.zip, where ?????? is the version number, based upon
date of modification.  There may also be some additional sound files
included that are not accessed by the Theme.

This zip also contains the file, logo.sys, which is the Windows 95 startup
image.  If you'd like to change to this logo, backup your copy of
logo.sys located in C:\, then copy logo.sys from the "Themes\Bullwinkle"
directory to C:\logo.sys.  If you'd merely like to view this file,
you can use MSPAINT or any other suitable program for viewing bitmaps (BMP).

I'm still working on this theme, with the only changes left to create some
Rocky and Bullwinkle animated cursors.  The cursors contained in this
version (960125) are ones easily found elsewhere on the Net.  I'm also
still working on the ff_icons_*.zip set...
