INSTALLATION: 
-------------
  1) Open the g1oprime.ZIP file in WinZip.
  2) Click on the EXTRACT button.
  3) This will open the Extract Window.  In the "Extract To:" box,
     select the directory that contains your themes.  Will probally
     be C:\PROGRAM FILES\PLUS!\THEMES
  4) In the "Files" box, select "All Files".
  5) Check "Use Folder Names".
  6) Click on the "Extract" button.  This will automatically
     create a Transformers - Optimus Prime directory in your themes directory
     and extract the files there.
  7) Load the theme with the Desktop Themes program.
