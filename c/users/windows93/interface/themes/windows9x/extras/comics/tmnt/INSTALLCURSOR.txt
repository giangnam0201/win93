
			INSTALLATION

TMNT CURSORS is distributed in a compressed archive file.  To
decompress it, you will need a utility such as PKZIPｮ for Windows or
WinZipｮ.  If you don't already have one of these programs, there are
shareware versions of both that can be downloaded over the Internet.
(http://www.winzip.com) Once you have a decompression utility, decompress 
the cursors into your Windowsｮ Cursors directory (normally 
C:\Windows\Cursors). 

			CONFIGURATION

Now that you've downloaded TMNT CURSORS and installed them, you must
let the computer know that you want to use them, instead of the default
Windowsｮ cursors.  Windowsｮ makes this easy by using the Mouse Properties 
box. 

1)	On the Start Menu select Settings then Control Panel. 

2)      Then double click on the Mouse icon in the Control Panel
        window.

        The Mouse Properties box will then pop up. 

3)	Click on the Pointers tab.  A menu will appear. This menu
        displays the cursors that you are currently using. 

4)	To change the Normal Select cursor, click on the cursor
	in the menu (which will highlight it), then hit the Browse
	button.  A box will then pop up, displaying the contents of
	your Windowsｮ Cursors directory. 

5)	Find and choose the filename tmnt-cur and then hit the Open 
	button. 

	This will replace your original cursor with the new TMNT
	cursor.  Repeat this process with the rest of the cursors,
	using the following filenames to replace the cursors. 

		Help Select		tmnt_help
	     	Working In Background	tmnt_work
		Busy			tmnt_busy
		Precision Select	tmnt_cross
		Text Select             tmnt_beam
		Handwriting             tmnt_write
		Unavailable             tmnt_no
		Vertical Resize         tmnt_ns
		Horizontal Resize       tmnt_we
		Diagonal Resize 1       tmnt_nwse
		Diagonal Resize 2       tmnt_nesw
		Move                    tmnt_move
		Alternate Select        tmnt_alt
		Link Select		tmnt_link

6)	After you have finished changing all the cursors, click on
	the Scheme Save As... button.  Name the scheme TMNT and hit OK.
	Now click the Apply button.  Your cursor should	change into the 
	new turtle cursor.  Click OK to close the Mouse Properties box. 

That's it!  You have just successfully changed your Windowsｮ 95 cursors
into TMNT CURSORS.  Enjoy! And have fun! Be like Donnie and keep 
discoing! :)
