Instructions:
Download this file and then use a program such as Winzip (go to 
http://www.winzip.com to download) or PKzip (go to http://www.pkware.com to 
download) to decompress it. If you do not have the PLUS! pack from Microsoft 
(http://www.microsoft.com) you can use other programs such as Desktop 
Themes, published by Left Side Software (go to 
http://www.lss.com.au/lss/windows/dt/dt_ins.htm to download it). Extract all 
of the files from NinjaTurtlesTheme.zip into their own folder in your PLUS! 
themes directory (c:\program files\plus!\themes\NinjaTurtlesTheme\) Make 
sure the files go in that directory, otherwise the file won't work. If you 
had the PLUS! program, then go to your Control Panel (click on start menu, 
click on settings, click on control panel) and double click on the Desktop 
Themes icon. If you Downloaded the Desktop Themes program, then you should 
have a shortcut in a folder named "Desktop Themes" on the programs menu of 
your start menu. Click on the "install" tab. You can select the 
NinjaTurtlesTheme theme from there, or you probably will have to click the 
"Browse" button and browse to the themes folder 
(c:\program files\plus!\themes) and select the NinjaTurtlesTheme.theme icon.
