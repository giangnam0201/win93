Theme Name: Apple II
Theme Description: This theme was created using Theme Studio 99. The wallpapers were
resized by using MS Paint, one being an 800x600, the other a 1024x768. The Apple II
icons were recreated using Icon Forge, with a few extra icons added. The cursors and
wav files are from another Macintosh theme from my collection.  

This what version 1.1 of an Apple II Desktop looked like in 1986; very primative looking
compared to the OSX of today, but during its day, Apple II sold in the millions and was
as popular and widely used then as Windows 98/2000/ME/XP is now.

Your computer should either have Microsoft Plus! or Desktop Architect in order for this 
theme to work properly.

Theme Installation for Windows 95/98/ME :
01. Create a theme folder using the following path:
             C:\Program Files\Plus!\Themes\Apple II
02. Unzip the theme files into the new theme folder.
03. Run the theme program; scroll down from "Current Window Settings" and select "other".
    Click on the "Apple II" folder, then click on desktop theme icon, or type in the file 
    name. Click on "Apply" and after it has installed, click "OK"
