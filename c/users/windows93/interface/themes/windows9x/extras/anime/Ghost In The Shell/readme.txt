Theme Installation:

Copy, cut and paste, or unzip the "Ghost in the Shell" 
folder and the "Ghost in the Shell.theme" file into your 
themes directory. Usually "C:\Program Files\Plus!\Themes"  

Double click on Desktop Themes in the Control Panel and 
apply the Ghost in the Shell theme.
