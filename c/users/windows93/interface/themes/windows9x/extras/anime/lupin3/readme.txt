Our names are Giovanni & Sergio Di Chiara. We are Lupin fans principalement of
1st  serie. This theme has done for a Windows 95 configured at 65,000 or more
colours (because the animated cursors are at 256 colours but they need 65,000
colours).
The opening song of Windows is the 2nd serie opening song, while all others
sounds are of 1st serie.
We have done also 2 versions of  logos.sys and logow.sys, we advise to copy
them in windows directory.

We have done also a Dragon Ball theme! In this Dragon Ball theme there are 15
animated cursors (very fancy) and a lot of icons.

If somebody have Lupin '68 song (sung by Corsay (that one used in the 1st
serie) (it isn't that one of  CD), can you write us, please?

"Dead or Alive" web page).

Italian version

Siamo Giovanni e Sergio Di Chiara, grandi fan di Lupin soprattutto della prima
serie.
I nostri cursori animati sono a 256 colori, ma per vederli bene il Windows 95
deve essere settato a 65000 colori.
Il wave di apertura del Windows 95 č la sigla iniziale della seconda serie,
mentre gli altri suoni sono presi dalla prima serie.
Abbiamo fatto due versioni dei file logow.sys e logos.sys, consigliamo di
copiarli nella directory del windows.
Abbiamo fatto anche un Dragon Ball theme, ha 15 cursori animati (molto curati)
e un sacco di icone.

Se qualcuno ha la canzone Lupin '68 (cantata da Corsay (per intenderci quella 
originale che si sente nella prima serie) ), puň perfavore mandarci una e-mail?
