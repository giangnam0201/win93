2 Installation
--------------
Copiez le fichier vers votre repertoire \windows\system (sous Windows 95)
ou \windows\system32 (sous Windows NT) et choisissez "electric" dans le
panneau de configuration.
Vous aurez de meilleurs resultats en configurant votre ecran en 256
couleurs.

4 Quoi de neuf dans la version 1.2 ?
------------------------------------
* Ajouté : Utilisation de DirectX pour changer le mode vidéo au lancement.
Pour l'instant ça ne marche pas bien sous Windows NT 4.0,
et ce n'est pas supporté par les versions antérieures de Windows NT.
* Ajouté : Possibilité de changer la vitesse d'animation.
* Ajouté : Possibilité de choisir la palette de couleurs.
* Ajouté : Prévisualisation en plein écran permattant de changer les options 'au vol'.
* Corrigé : L'option 'Screen scan' du dévéroleur McAffee n'arrête plus nécessairement l'économiseur.

6 Bogues et limitations connues
-------------------------------
En mode 16 couleurs :
* Le résultat est vraiment _moche_. A votre place, j'utiliserais une autre
résolution ...
En mode 256 couleurs : 
* Il paraît (?) que ce programme exerce une fascination étrange sur les _chats_.
Bon, moi je n'ai pas de chat, je me contente de répéter ce qu'on a pu me dire,
mais si votre matou devient dingue ne venez pas me dire que vous n'avez pas été prévenu.

En mode 65536 couleurs (et plus) :
* L'image est nette, mais tout ce qu'il y a de plus fixe. L'animation de palette est une
possibilité qui n'est offerte par le matériel qu'en mode 256 couleurs.

Avec DirectX
* L'option DirectX a de fortes chances de planter Electric! sous Windows NT 4.0. A vos risques et périls.
* L'option 'Screen scan' du dévéroleur McAffee peut provoquer un retour au mode vidéo par défaut.
Je cherche une solution...

Avec Windows NT
* L'option 'ignorer les mouvements de la souris' ne fonctionne pas sous NT 4.0.
* La boîte de configuration répond lentement sous NT 3.51.
* La boîte de configuration n'apparaît pas sous Windows NT 3.1 à 3.5.
