2 Installation
--------------
Wenn Sie das Programm als Archiv (Endung ".zip") erhalten
haben, mit pkunzip entpacken; einfach die Datei "electric.scr"
in Ihr System-Verzeichnis (z.B. c:\windows\system oder
c:\windows\system32 fuer NT) verschieben. Dann ueber die Sys-
temsteuerung Electric als Screensaver auswaehlen.

6 Bekannte Probleme
-------------------
16-Farben-Modus:
* Das Ergebnis sieht einfach, schlicht und ergreifend gesagt,
  scheusslich aus. An Ihrer Stelle wuerde ich einfach eine
  andere Farbtiefe waehlen...

256-Farben-Modus:
* Mir wurde zugetragen, das Programm uebe eine seltsame Faszi-
  nation auf Katzen aus. Kein Witz. Da ich aber keine habe,
  kann ich nur wiedergeben, was mir erzaehlt wurde. Aber wenn
  Ihre Mieze durchdreht, erzaehlen Sie mir nicht, ich haette
  Sie nicht gewarnt...

65536 Farben und mehr:
* Die Farbuebergaenge sind sehr sanft, aber das Bild ist nicht
  animiert. Das sogenannte "Color Cycling" wird unter diesen
  Farbtiefen leider nicht unterstuetzt.

Unter DirectX:
* Unter Windows NT 4.0 kann die DirectX-Option einen Crash ver-
  ursachen; bitte nicht verwenden.
* Die McAffee Screen-Scanner-Option kann ein Zuruecksetzen des
  Farbmodus auf die Voreinstellungen verursachen. Ich muss erst
  noch herausfinden, wie man dies beheben kann.

Unter Windows NT:
* Die 'ignore mouse moves'-Option arbeitet nicht korrekt unter
  Windows NT 4.0.
* Der Einstellungs-Dialog ist unter Windows NT 3.51 sehr, sehr
  langsam.
* Der Einstellungs-Dialog funktioniert nicht unter Windows NT
  Version 3.1 bis 3.5.
