problems with unziping ??

Most of this theme has long filenames, which of course means that the zip 
archive must be unzipped with a program that supports these long filenames. 
Using an old unzip program will result in broken long filenames which make it 
impossible to install the theme. WinZip from Niko Mak Computing, version 6.0 
for Windows 95 or newer is recommended. If you don't have WinZip or if you 
have an older version, you can download an evaluation version or update from 
their web site at <http://www.winzip.com/>.
