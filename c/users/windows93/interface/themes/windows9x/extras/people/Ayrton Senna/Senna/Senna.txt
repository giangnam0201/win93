Hello.  Thank you for trying out this theme.

Just unzip the Senna Theme.ZIP file (don't forget to use the
directory names) and enjoy!  You must use a zip program that
supports long filenames (i.e. WinZip).

The screen saver will need to be installed separately, there is a
rainss.zip, extract it somewhere out of the way and run the
setup.exe. 

Its not a brilliant screen saver, but thought that rain would
appropriate, as these were the driving conditions in with Senna
excelled.

This is my first attempt at a Theme, and I would like to improve on
it, so if you have a better screen saver, sound effects, piccies etc
mail them to me and I will try to use them.
