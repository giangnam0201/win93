This Theme is my tribute to the greatest martial artist that ever lived!
Bruce Lee - the little dragon.

I have been searching the net for several months for a Bruce Lee theme and was surprised when there was no theme available to the millions of fans around the world.
So, I decided to make my own.

Many thanks to the creators of the various files that I compiled to make this theme.

To setup the Bruce Lee theme.
Unzip with Winzip to  C:\Program Files\Plus!\Themes\Bruce Lee\
setup the theme as you ussually would.

Move the fullmotn.scr saver to the C:Windows\System directory 

To activate the Bruce Lee Movie file as a screen saver do the following:
 	1. Click the windows start button.
	2. Go to settings.
	3. Click on control panel.
	4. Click on display.
	5. Clicl on the Screen Saver tab.
	6. Choose full Motion as the screen saver.
	7. Click on settings.
	8. Click Audio on and then Click select main videos for playback.
	9. using the directory find the lee.mpg file in the Bruce Lee Directory.
	10. Select the video, save and exit out of settings.
	11. Click preview to test.
	
After everything is set up, reboot and Enjoy!
