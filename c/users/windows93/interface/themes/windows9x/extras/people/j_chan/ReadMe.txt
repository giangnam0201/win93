How to install the theme:
------------------------------

You must unzip the package in your Themes directory 
The Screensaver, the dll's and the sys files must be
placed in the windows directory. You should backup the
original sys files!!!

ATTENTION PLEASE:
Before unzipping with WINZIP make sure the "Use Folder Names" box is checked because 
WINZIP makes the directory "Jackie Chan 2.0" to your Themes directory.
