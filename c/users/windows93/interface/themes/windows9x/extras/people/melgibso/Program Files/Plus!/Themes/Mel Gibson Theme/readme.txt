Mel Gibson Theme

Created by Nancy Martin (martinna@cadvision.com)

This is my first attempt at making a theme.  I hope you enjoy it.  This theme is free.  If you enjoy it please e-mail me and let me know.

To best view this theme, make sure that under "Display" in Control Panel, you have your color palette set to "High Color (16 bit)".

To install the Mel Gibson Theme:

1.  Unzip the contents into your C:\ directory.  Winzip will place the files in the appropriate folders.

2.  Go to Control Panel, Desktop Themes, and select the Mel Gibson (high color) theme.

3.  Make sure that you do not have "stretch desktop wallpaper to fit screen" toggled on.  To do this go to Control Panel, Display, click the Plus! tab and clear the box for this option.

Enjoy!!!
