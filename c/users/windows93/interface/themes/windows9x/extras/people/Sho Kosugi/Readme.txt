Introduction
~~~~~~~~~~~~
Thank you for downloading my theme, I hope you enjoy using it. All materials in the theme were gathered from the web or was created by me. 
This theme is dedicated to the legendary Martial Artist and Actor Sho Kosugi, who introduced and defined the image of the Ninja for movie fans all over the world. When Ninja films in the 80`s were the craze and when Sho Kosugi was known as the "Master Ninja".


Requirements
~~~~~~~~~~~~
Windows 98®
Microsoft Plus!®
MPEG Layer-3 Sound Support



Sound Information
~~~~~~~~~~~~~~~~~
Sound drivers for MPEG Layer-3 MUST be installed. If you can´t hear the sound files, you may need to install Microsoft Netshow Version 2.0, available at: http://msdownload.microsoft.com/msdownload/netshow/2.0/x86/en/nsplay.exe or if you have Win 98 you can simply install the "Audio compression" in the Control Panel/ Add/Remove Programs/ Multimedia, and then click details. Select Audio Compression and Microsoft NetShow Player 2.0.


Install
~~~~~~~
1. Unzip theme to your desired destination.
2. Copy the font file to your windows font directory.
3. Use Microsoft Plus! or your choice of third party theme managers to apply your theme settings.

Uninstall
~~~~~~~~~
Delete the theme and zip file.
