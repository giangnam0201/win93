Unzipping issues
================

Most of the theme's files have long filenames, which means that the
archive must be unzipped with a program that supports these long
filenames. Using an old unzip program results in broken long filenames
which make it impossible to install the theme. I recommend using WinZip
from Niko Mak Computing, version 6.0 for Windows 95 or newer. If you don't
have WinZip or if you have an older version you can download an update or
evaluation version from their web site at <http://www.winzip.com/>.

After having unzipped the archive, you should have eleven files and one
folder. The folder and some of the files should have long filenames. If
you only have files with short names or if your unzip program generated a
lot of error messages, you have used a program that can't handle long
filenames.

If you are using a Windows 95 unzip program but you still have problems
getting the archive unzipped correctly you should verify that your unzip
program uses "internal unzipping", meaning that it does not run the
MS-Dos version of PkUnzip to do the actual unzipping.
