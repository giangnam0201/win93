To install the Frank Zappa themes:

You should unzip this package with a program such as Winzip32 that can
handle win95 long file names.


1)Create a folder called "Frank Zappa" under your "Themes" directory.
  It probably will look like this: 

		c:\Program Files\Plus!\Themes\Frank Zappa

2)Unzip all of the files into your "Frank Zappa" folder EXCEPT for the
  "*.the" files. Place these files into your "Themes" folder. 

  *By creating a new folder under your "Themes" folder for each new theme,
   you minimize the amount of garbage in your "Themes" folder and it 
   is also easier to delete the contents of the theme if you think it stinks.

3) Go into your control panel and select either of the Zappa themes.
   They are the same I just threw in a few different wallpapers.
