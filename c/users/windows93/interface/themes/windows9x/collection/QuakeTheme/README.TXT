Introduction

Quake - The Theme 1.0 is a desktop theme designed for Microsoft Plus! for Windows95.
This theme includes

	Completely original animated icons
	Authentic sounds from the hit game Quake
	A totaly original screen saver
	And much much more!!!

Requirements

Well as of this version, it's recommended that you have at least 16-bit color and a high
resolution video card. (We figured if you were playing Quake you would have these things
anyways.  Updates to this theme will be available via the Official Quake Theme homepage!



Archive

Place all files in your Themes directory with THESE EXCEPTIONS!!!!!!

---------------------
Theme Directory Files
---------------------

Place all files in your themes directory.
Usually C:\Program Files\Plus!\Themes

---------------------
Fonts
---------------------
Add Fonts from the Font directory through Control Panel/Fonts.

---------------------
Screen Saver
---------------------
Place the files from the Screen Saver directory in 
C:\Windows\System directory.

olepro32.dll		C:\Windows\System	OLE for 32 bit        --|
vb40032.dll		C:\Windows\System	Visual Basic Runtime  --|
comctl32.ocx		C:\Windows\System	VB Custom Control     --|
quake.scr		C:\Windows\System	Quake Screen Saver!   --|
