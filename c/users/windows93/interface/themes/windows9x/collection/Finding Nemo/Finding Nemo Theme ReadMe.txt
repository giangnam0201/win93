Finding Nemo Theme.
Created with Desktop Architect

This theme contains Wallpaper in 1024X768 format (set to stretch to fit screen),
Original Icons, Cursors, System Logo Screens, Webview Images, Sounds,
and an Original Screensaver.

To install, unzip the contents directly to your themes directory
(i.e. c:\Program Files\Plus!\Themes\ or wherever your themes are stored).
Everything will go where it belongs and the proper subdirectories will
be created. 

To install the screensaver, double click the Setup Finding Nemo.exe icon.
It will automatically install and be set as your default screensaver.

I hope you enjoy this theme.
