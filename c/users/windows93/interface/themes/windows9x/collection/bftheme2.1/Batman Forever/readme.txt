Batman Forever (High Color) Theme for Microsoft Plus Pack version 2.0

This is for all those Batman Forever fans.

This is my first attempt at doing this and I know that there is a lot of improvement that can be made to this. 

I still don't know how to make a screensaver therefore this theme does not contain a screensaver. You can use one of the standard Windows screensavers.

Feel free to distribute this theme. If you do make changes I would appreciate if you could let me know. This is a Shareware theme, but it does not have any annoying screens
to bug you. I would appreciate any donations from people who decide to use this theme. It would pay for my pizza and help me make improvements to this and other themes that 
I am working on.

INSTALLATION
Unzip the files to a directrory called Batman in the Themes* directory. If you ever want to uninstall, just delete the Batman directory.

* if you use Winzip to unzip the files to the the Plus! directory, the files will automatically extract to this directory. The directory is created automatically.
