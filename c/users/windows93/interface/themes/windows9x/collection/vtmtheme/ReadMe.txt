The Vampire: The Masquerade Windows
95 Microsoft Plus! Theme
ReadMe File

Instructions for Plus! Users
·	Make sure you have the "Mystery" theme installed.  This theme uses the
Screen Saver and Sound Clips from that theme.  "Mystery" is included
with Plus!  If you don't give  a flying foo about Screen Savers and/or
Sound Clips, than you don't need "Mystery."
·	UnZIP The File's Contents into your Themes directory (probably
C:\Program Files\Plus!\Themes)
·	In Control Panel, double-click the Fonts applet.
·	Under the File menu, click "Install new font"
·	Once the Install Font window appears, navigate to your Themes directory
and install all the fonts you find there.
·	Close the Fonts Window
·	Double-Click the Plus! Themes applet in Control Panel
·	In the bar-box at the top of the Themes Window, click the arrow and
scroll down to "Vampire The Masquerade (High Color) Theme" . Select.
·	Click OK and you're done!

Instructions for NON-PLUS! users
·	Download the "Dangerous Creatures Theme for Windows 95" demo from
Microsoft.  (http://www.microsoft.com)  It's on the Microsoft Download
page.
·	Install Dangerous Creatures
·	UnZIP the Vampire theme ZIP into the newly created "Program
Files\Plus!\Themes" Folder.
·	In Control Panel, double-click the Fonts applet.
·	Under the File menu, click "Install new font"
·	Once the Install Font window appears, navigate to your Themes directory
and install all the fonts you find there.
·	Close the Fonts Window
·	Close Control Panel
·	Navigate, using Windows Explorer, to "C:\Program Files\Plus!\Themes"
·	Once in that folder, click on the Dangerous Creatures Theme's name. 
Click on it again, after a few seconds, so that the computer simply
highlights the name as if you were going to rename it.  Right-click on the
highlighted name and COPY it.  Then rename the file to something else. 
Doesn't matter what, as long as it's not what it was.
·	Click on the Vampire The Masquerade (High Color) Theme's name. 
Click on it again, after a few seconds, so that the computer simply
highlights the name as if you were going to rename it.  Right-click on the
highlighted name and PASTE.  Tap the Enter key.
·	Now, run the Dangerous Creatures theme Demo.  In the bar-box at the top
of the Themes Window, click the arrow and scroll down to Dangerous
Creatures . Select.
·	Click OK and you're done!
