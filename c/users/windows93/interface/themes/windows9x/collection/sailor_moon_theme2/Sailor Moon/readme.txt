Well, with all that out of the way, thanks for downloading the first-ever
(un)Offical Sailor Moon Theme Pack for Windows 95. You will need to have
Windows 95 and the Windows 95 Plus Pack already installed on your machine.
You will also need an additional 1MB of disk space to install this theme
(this should be the easy part).

I put all this together out of withdrawl from loosing Sailor Moon in NA.
When I started watching my tapes all over again, I just started recording

Installation:
Just use a Windows 95 native 32-bit un-zipping program (like WinZip).
Unzipping this file into your root directory (ie. C:\) should put all the
files into the correct locations. It will make a directory in your theme
directory called 'Sailor Moon' (that was obvious, huh??). Go to your Control
Pannels (Start--Settings--Control Panel) and run the Desktop Themes control
panel. Choose the theme called 'Sailor Moon (True Color)'. That's it!!

Manual Installation:
What, that didn't work? Well, here's the information you need to install the
pack manually. Install ALL the files into a new folder called 'Sailor Moon'
(..uhhh...without the quotes) in your 'Themes' folder; that should be
C:\Program Files\Plus!\Themes. Then copy (or move) the file called 'Sailor
Moon (High Color).theme' into your theme folder. Now you should be able to
find it listed in your Desktop Themes control panel.

Assorted Ramblings:
I have the screen saver set to the Science screen saver. If that's not
installed on your system, you'll have to set one. The destop picture I have
included with this theme is a 640x480 image with 256 colors (...ok, I found
it on the net). It should look OK at most reslotions.

All of the icons and cursors were done in high color mode. Sorry, I HATE
having to torture myself with 256 colors or less. So, if you're running 256
colors (or, Heaven forbid, 16 colors), you'll see some really yucky looking
icons. If you're one of those who run Windows at 256 colors, increase it.
What?? You can't?? Do you have the latest video drivers for your system??
You're on the internet - you could go find some updated drivers. Chances are
good that this should not be a problem.
